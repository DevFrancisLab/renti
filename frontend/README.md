# Renti Frontend

This repository contains the frontend application for the Renti property management system.

### Development

1. Install dependencies:
	```sh
	cd frontend
	npm install
	```

2. Run in development mode:
	```sh
	npm run dev
	```

3. Build for production:
	```sh
	npm run build
	```

### Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Deployment

Deploy to your hosting provider of choice (e.g. Vercel, Netlify). Serve the `dist` folder after building.

### Custom Domain

Set up a custom domain through your hosting provider's configuration.
